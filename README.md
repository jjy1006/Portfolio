### 오픈뱅킹서비스 프로젝트 설명
이 프로젝트는 고객들이 은행 서비스를 온라인으로 이용할 수 있는 사이트를 제공하며, 타행 은행 계좌 송금을 가능하게 하여 하나의 사이트에서 여러 은행 계좌를 관리할 수 있도록 하는 것을 목표로 했습니다.

### 세부 기능
- 회원가입 및 계좌 개설
- Transaction 처리를 통한 당행 계좌 조회/이체 및 오픈뱅킹
